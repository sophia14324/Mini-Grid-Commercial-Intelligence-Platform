# Mini Grid Commercial Intelligence Platform

## Business problem and value
A mini-grid operator needs to answer 5 commercial questions from its data:

* Which sites consume the most energy
* Which appliances drive peak demand
* Which sites have voltage anomalies that signal equipment problems
* What revenue is generated under flat tariff versus a time-of-use(TOU) model
* Which sites have stale or missing data that could indicate meter failures

These are not toy questions. Mini-grid operators use exactly this logic to allocate maintenance resources, design tariff structures, and forecast grid capacity.
